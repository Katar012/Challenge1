# PWA Challenge 2

Link del proyecto: [https://challengemovil.netlify.app/](https://challengemovil.netlify.app/)

### **Android (Chrome)**
1.  Abre **Chrome** e ingresa a [**challengemovil.netlify.app**](https://challengemovil.netlify.app/).
2.  Espera unos segundos a que aparezca el aviso: **"Agregar a la pantalla de inicio"**.
3.  Si no aparece:
    * Toca los **tres puntos** (menú) en la esquina superior derecha.
    * Selecciona **Instalar aplicación** o **Instalar**.

### **iOS (Safari)**
1.  Abre **Safari** e ingresa a [**challengemovil.netlify.app**](https://challengemovil.netlify.app/).
2.  Toca el icono de **Compartir** (el cuadrado con una flecha hacia arriba).
3.  Desliza hacia abajo y selecciona **Agregar a la pantalla de inicio**.
4.  Confirma tocando **Agregar**.
